Notifications iPhone

Pour des notifications fiables sur iPhone, utilise l'app Raccourcis.
Crée deux automations qui s'exécutent automatiquement :

🌅 Matin — rappel arrivée

1. Ouvre Raccourcis → onglet Automatisations

2. Appuie sur + → Heure de la journée

3. Heure : 08:00 — Répéter : Lun→Ven

4. Action : Afficher la notification

5. Titre : ⏰ Pointer arrivée !

6. Corps : N'oublie pas de pointer ton arrivée

🌆 Soir — rappel départ

1. Même étapes → Heure : 17:00 — Répéter : Lun→Ven

2. Titre : 🏠 Pointer départ !

3. Corps : N'oublie pas de pointer ton départ
